# ESP32_Projects

This repository contains my ESP32 projects using Arduino IDE.

## Projects

- **Hello_esp32** → Prints HELLO ESP32 to the Serial Monitor.
- **BLUE_LED_on** → Turns the onboard blue LED ON (GPIO2).
- **LED_task** → Turns led 1 on off every 500ms and led 2 every 1000ms using Task.
- **summer_day** → Turns led on if temperature exceeds 25degree celcius.
- **Realtime_thrmo_gyrometer** → dht11 senses temperature and humidity every 2 seconds and print it on lcd screen(the screen is updated every 1 second).
